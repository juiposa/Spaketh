# Spaketh

---

### A FFXIV Plugin for having voice lines play over unvoiced bosses. 

This plugin takes the voicelines of voiced battle NPCs and lays plays them in place of the speech of unvoiced battle NPCs.

The first supported boss and the POC of this is Alexander Resurrected (Windhurt) -> Alexander (TEA, A12).

#### Commands

- `/spaketh` - quick enable/disable the plugin's functionality
